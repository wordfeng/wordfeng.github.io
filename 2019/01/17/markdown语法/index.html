    10.流程图

    我这个是GitHub存储的，GitHub不支持流程图，模板用的hexo，默认也不支持流程图，但是hexo可以用插件强行支持。
    npm install --save hexo-filter-flowchart
    反引号反引号反引号(由于渲染问题，我就用这个代替三个`了)flow
    —-定义部分格式： 变量名=>模块名: 内容—-
    st=>start: 开始
    op=>operation: My Operation
    cond=>condition: Yes or No?
    e=>end
    —-流程控制部分格式：变量->变量—-
    st->op->cond
    cond(yes)->e —cond(yes)表示判断为yes执行的路线
    cond(no)->op
    反引号反引号反引号

    操作模块 说明
    start 开始
    end 结束
    opration 普通操作块
    condition 判断块
    subroutine 子任务块
    inputoutput 输入输出块
